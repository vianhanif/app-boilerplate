## Tools used
- Create React App (1.0.14)
- React (16.0.0 - "Fiber")
- Redux (3.7.2)
- Thunk (2.2.0)
- React Helmet (5.2.0)
- React Router v4
- React Router Redux (5.0.0-alpha.6)
